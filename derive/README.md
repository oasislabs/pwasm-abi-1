# owasm-abi-derive
Derivation macros for traits to bind dispatch and invoke using `owasm-abi`.

[Documentation for original `pwasm-abi`](https://paritytech.github.io/pwasm-abi/pwasm_abi/)

# Add dependency

```toml
[dependencies]
owasm-abi-derive = "0.2"
```
